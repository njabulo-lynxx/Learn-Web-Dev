Lists in HTML

HTML provides several types of lists to organize content in a structured manner.
The most commonly used list types are ordered lists, unordered lists, and definition lists.

1. Ordered Lists

Ordered lists are used when the order of items matters. They are created using the <ol> tag, with each item defined by the <li> tag.

Example:

<ol>
  <li>Item 1</li>
  <li>Item 2</li>
</ol>

2. Unordered Lists

Unordered lists are used when the order of items does not matter. They are created using the <ul> tag, with each item defined by the <li> tag.

Example:

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

3. Definition Lists

Definition lists are used to define terms and their definitions. They are created using the <dl> tag, with each term defined by the <dt> tag and
each definition by the <dd> tag.

Example:

<dl>
  <dt>Term</dt>
  <dd>Definition</dd>
</dl>

By using these list types, you can effectively organize and present information in your HTML documents.
