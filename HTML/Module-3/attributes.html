HTML Attributes

Attributes provide additional information about HTML elements. They are always included in the opening tag and usually come in name/value pairs like name="value".

Common HTML Attributes

  • id: Specifies a unique id for an element. Example: <div id="header"></div>
  • class: Specifies one or more class names for an element (used for CSS and JavaScript). Example: <p class="intro"></p>
  • src: Specifies the source URL for media elements like images. Example: <img src="image.jpg" alt="Description">
  • href: Specifies the URL for a link. Example: <a href="https://www.example.com">Visit Example</a>
  • alt: Provides alternative text for images. Example: <img src="image.jpg" alt="A beautiful scenery">
  • title: Provides additional information about an element (displayed as a tooltip). Example: <abbr title="Hypertext Markup Language">HTML</abbr>
