Creating a Drop-down Menu in HTML

A drop-down menu allows users to select one option from a list. You can create a simple drop-down menu using the <select> and <option> elements in HTML.

Example of a Drop-down Menu

Explanation

  • The <select> element creates the drop-down menu.
  • The id attribute is used to uniquely identify the drop-down menu.
  • The name attribute is used to reference the selected value when the form is submitted.
  • Each <option> element represents an item in the drop-down list. The value attribute specifies the value that will be sent to the server when that option is selected.
