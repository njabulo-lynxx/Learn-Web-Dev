HTML Form Basics

Forms are used to collect user input. They can contain various types of input elements like text fields, checkboxes, radio buttons, and submit buttons.

Creating a Simple Form








Common Form Elements

  • <input>: Used for various types of user input (text, password, checkbox, radio, etc.). Example: <input type="text" name="username">
  • <textarea>: Used for multi-line text input. Example: <textarea name="comments"></textarea>
  • <select>: Used to create a drop-down list. Example: <select name="options"> <option value="option1">Option 1</option> <option value="option2">Option 2</option> </select>
  • <button>: Used to create clickable buttons. Example: <button type="submit">Send</button>

Other Useful Form Elements

  • <fieldset>: Groups related elements within a form. Example: <fieldset> <legend>Personal Information</legend> <input type="text" name="firstname"> <input type="text" name="lastname"> </fieldset>
  • <legend>: Provides a caption for the <fieldset> element. Example: <legend>Contact Details</legend>

Input Types

  • text: Single-line text input. Example: <input type="text" name="firstname">
  • password: Password input (characters are hidden). Example: <input type="password" name="pwd">
  • email: Input for email addresses. Example: <input type="email" name="email">
  • checkbox: Checkbox input. Example: <input type="checkbox" name="subscribe">
  • radio: Radio button input. Example: <input type="radio" name="gender" value="male">
  • submit: Submit button to send form data. Example: <input type="submit" value="Submit">
  • reset: Reset button to clear all form fields. Example: <input type="reset" value="Reset">
  • file: Input for file uploads. Example: <input type="file" name="upload">
  • date: Input for selecting a date. Example: <input type="date" name="bday">
  • number: Input for numeric values. Example: <input type="number" name="quantity">
  • url: Input for URLs. Example: <input type="url" name="website">
  • tel: Input for telephone numbers. Example: <input type="tel" name="phone">
  • color: Input for selecting a color. Example: <input type="color" name="favcolor">
  • range: Input for selecting a value within a range. Example: <input type="range" name="volume" min="0" max="100">
  • hidden: Hidden input field. Example: <input type="hidden" name="userid" value="12345">
  • search: Input for search queries. Example: <input type="search" name="query">
  • time: Input for selecting a time. Example: <input type="time" name="appt">
  • week: Input for selecting a week. Example: <input type="week" name="weeknum">
  • month: Input for selecting a month. Example: <input type="month" name="bmonth">
  • datetime-local: Input for selecting a date and time. Example: <input type="datetime-local" name="meeting">
  • button: Generic button. Example: <input type="button" value="Click Me">
  • image: Image as a submit button. Example: <input type="image" src="submit.png" alt="Submit">

Form Attributes

  • action: Specifies where to send the form data when submitted. Example: <form action="/submit">
  • method: Specifies the HTTP method to use when sending form data (GET or POST). Example: <form method="post">
  • name: Specifies a name for the form, which can be used to reference it in scripts. Example: <form name="contactForm">
  • autocomplete: Specifies whether the browser should autocomplete input fields. Example: <form autocomplete="on">

Form Labels

Labels are used to provide descriptions for form elements. They improve accessibility and usability.

<label for="elementID">Description:</label>
        <input type="text" id="elementID" name="elementName">
