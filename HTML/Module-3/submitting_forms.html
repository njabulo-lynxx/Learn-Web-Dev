Submitting Forms

When a user fills out a form and clicks the submit button, the form data is sent to a server for processing. The way the data is sent depends on the method attribute of the <form> element.

Form Submission Methods

  • GET: Appends the form data to the URL in name/value pairs. This method is suitable for non-sensitive data and has length limitations. Example: <form action="/submit" method="get">
  • POST: Sends the form data as part of the HTTP request body. This method is more secure for sensitive data and does not have length limitations. Example: <form action="/submit" method="post">

Form Action Attribute

The action attribute of the <form> element specifies the URL where the form data should be sent for processing. If the action attribute is omitted, the form data is sent to the same page that contains the form.
