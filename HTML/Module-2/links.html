Creating Hyperlinks in HTML

Hyperlinks are an essential part of the web, allowing users to navigate between different pages and resources. In HTML, hyperlinks are created using the <a> (anchor) tag.

1. Basic Hyperlink

To create a basic hyperlink, use the <a> tag with the href attribute, which specifies the URL of the page the link goes to. For example:

<a href="https://www.example.com">Visit Example.com</a>

2. Linking to Different Pages

You can link to different pages within the same website by using relative URLs. For example:

<a href="/about.html">About Us</a>

3. Opening Links in a New Tab

To open a link in a new tab or window, use the target="_blank" attribute. For example:

<a href="https://www.example.com" target="_blank">Visit Example.com</a>
