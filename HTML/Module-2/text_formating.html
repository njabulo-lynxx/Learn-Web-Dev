Text Formatting in HTML

HTML provides various tags to format text and enhance the presentation of your content.
Here are some commonly used text formatting tags:

1. Headings

Headings are defined using the <h1> to <h6> tags, with <h1> being the highest level and <h6> the lowest.

2. Paragraphs

The <p> tag is used to define a paragraph of text.

3. Bold and Italic Text

You can make text bold using the <strong> or <b> tags, and italic using the <em> or <i> tags.

- The <strong> tag is semantically stronger than the <b> tag, while <em> tag is semantically stronger than the <i> tag.

- The <strong> tag is used to highlight important text.

- The <em> tag is used to define emphasized text.

4. Underlined Text

The <u> tag is used to underline text.

5. Strikethrough Text

The <s> or <del> tags are used to indicate text that has been deleted or is no longer relevant.

6. Superscript and Subscript

The <sup> tag is used for superscript text, while the <sub> tag is used for subscript text.

7. Line Breaks and Horizontal Rules

The <br> tag is used to insert a line break, and the <hr> tag creates a horizontal rule to separate content.

By using these text formatting tags, you can improve the readability and visual appeal of your HTML documents.
