HTML Menus and Navigation

Menus and navigation in HTML are created using lists and links. This section will cover how to create menus and navigation bars using HTML elements.

Creating a Simple Navigation Menu

A simple navigation menu can be created using an unordered list (<ul>) with list items (<li>) containing links (<a>). Here is an example:

<nav>
            <ul>
            <li><a href="index.html">Home</a></li>
            <li><a href="about.html">About</a></li>
            <li><a href="services.html">Services</a></li>
            <li><a href="contact.html">Contact</a></li>
            </ul>
            </nav>

Using the <nav> Element

The <nav> element is a semantic HTML5 element used to define a block of navigation links. It helps improve the accessibility and SEO of your website. Here is how to use it:

<nav>
            <ul>
            <li><a href="index.html">Home</a></li>
            <li><a href="about.html">About</a></li>
            <li><a href="services.html">Services</a></li>
            <li><a href="contact.html">Contact</a></li>
            </ul>
            </nav>
