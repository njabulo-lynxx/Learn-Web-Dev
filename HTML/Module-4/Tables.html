HTML Tables

Tables are used to display data in a structured format. They consist of rows and columns, and can include headers, footers, and captions.

Basic Table Structure

Name Age City
Alice 30 New York
Bob 25 Los Angeles

Table Elements

  • <table>: Defines the table.
  • <tr>: Defines a table row.
  • <th>: Defines a table header cell (bold and centered by default).
  • <td>: Defines a table data cell.
  • <caption>: Provides a title for the table.
