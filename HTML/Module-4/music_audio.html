Adding Audio to Web Pages

You can use the <audio> element to embed sound content in your web pages. The <audio> element supports various audio formats, including MP3, WAV, and Ogg.

Basic Usage

To add an audio file to your web page, you can use the following syntax:

<audio controls>
  <source src="audiofile.mp3" type="audio/mpeg">
  <source src="audiofile.ogg" type="audio/ogg">
  Your browser does not support the audio element.
</audio>

Attributes

  • controls: Adds audio controls like play, pause, and volume.
  • autoplay: Automatically starts playing the audio when the page loads.
  • loop: Repeats the audio indefinitely.
  • muted: Mutes the audio by default.

Example

Here is an example of how to use the <audio> element:

<audio controls autoplay loop>
    <source src="music.mp3" type="audio/mpeg">
    <source src="music.ogg" type="audio/ogg">
    Your browser does not support the audio element.
</audio>
