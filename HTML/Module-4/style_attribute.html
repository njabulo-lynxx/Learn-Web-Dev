The style attribute

The style attribute is used to apply inline CSS styles to an HTML element. It allows you to specify CSS properties and values directly within the HTML tag. The syntax for using the style attribute is as follows:

<element style="property: value;">Content</element>

For example, if you want to change the text color of a paragraph to red, you can use the style attribute like this:

<p style="color: red;">This text will be red.</p>

You can also apply multiple CSS properties by separating them with a semicolon. For instance, to set the background color to yellow and the font size to 20 pixels, you can write:

<div style="background-color: yellow; font-size: 20px;">This div has a yellow background and larger text.</div>

While the style attribute is useful for quick and specific styling, it is generally recommended to use external or internal CSS for better maintainability and separation of concerns in larger projects.
